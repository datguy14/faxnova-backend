# FaxNova Backend
A lightweight, production-ready fax delivery API built with Node.js, Express, and Twilio.

FaxNova Backend provides a clean and minimal API for sending faxes through Twilio’s Fax API.  
The architecture is intentionally simple, making it easy to deploy, maintain, and hand off to future owners.

---

## Features

- Send faxes via Twilio with a single REST endpoint
- Clean separation of routes, controllers, and utilities
- Minimal dependencies for fast startup and low overhead
- Environment-variable driven configuration
- Ready for deployment on Render, Railway, or any Node hosting platform
- MIT-licensed and acquisition-friendly
